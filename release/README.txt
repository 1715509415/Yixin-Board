English Version

System Requirements:
512MB RAM/32-bit video display adapter(True color)

FAQ:
Q: How to change the size of board?
A: Change the first line of "settings.txt" with the new size of board you want.
Q: How to change the language of GUI?
A: You can change the second line of "settings.txt"(It only supports English and Chinese now).

File List:
Yixin.exe
	The main program.
engine.exe
	The engine.
settings.txt
	The setting file of the main program.
base.dat
	An important file related to opening book.
bookX_Y.dat
	Opening book. X is the rule(0:freestyle, 1:standard, 2:free renju) and Y is the board size.
icon.ico
	The icon.
README.txt
	The README file.
piece.bmp
	Default board picture.
piece_N.bmp
	Board picture for NxN board.

运行环境要求：

常见问题：
问：如何改变棋盘大小？
